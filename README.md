# ✨ 魔法笔记 - Magic Note

**一本有灵性的魔法笔记，每当你在上面写字，它就会用古老的智慧回应你。**

就像《哈利波特》里的魔法笔记，又像《死亡笔记》那样神秘...但这是温暖的，充满智慧的魔法。

## 🌟 核心体验

- **✍️  沉浸式书写** - 深色星空背景，打字时的魔法粒子效果
- **✨ 自动感应** - 停止输入2秒后自动触发AI回应
- **🔮 魔法般的回复** - AI用神秘、温暖、富有哲理的语气回应
- **📖 笔记历史** - 所有对话自动保存，随时回看

## 🎨 界面特点

```
🌙 深色魔法主题
💜 深紫色神秘色调
✨ 背景流动粒子动画
🔮 发光边框效果
📜 卡片渐隐渐显动画
```

## 🤖 内置双AI模型

| 模型 | 特点 |
|------|------|
| **GLM-4-Flash** | 智谱AI，响应速度快 |
| **Moonshot** | 月之暗面，长上下文支持 |

API已内置，开箱即用！

## 📱 核心功能

### 1. 魔法书写页
- 大段文字输入区域
- 打字时显示魔法粒子效果
- 停止输入2秒后自动触发AI
- AI思考时显示旋转魔法光环
- 回复卡片带渐变和发光效果

### 2. 历史笔记页
- 按时间倒序排列
- 左滑删除笔记
- 点击查看完整内容
- 显示记录条数

## 🚀 快速开始


### 方式二：手动运行

```bash

flutter create .

# 2. 安装依赖

flutter run
```

## 💡 使用技巧

1. **写下心事** - 写下你的烦恼、困惑，魔法笔记会给你温暖的回应
2. **提问求答** - 问问题，笔记会用智慧的方式回答你
3. **记录感悟** - 写下你的感悟，笔记会给你更深的启发
4. **耐心等待** - 魔法需要时间显现，停止输入后等2秒
5. **回看历史** - 在「笔记」页查看所有对话记录

## 🔧 自定义AI回复风格

编辑 `lib/screens/magic_note_screen.dart` 中的 `_buildMagicPrompt` 方法：

```dart
String _buildMagicPrompt(String input) {
  return '''
你是一本有灵性的古老魔法笔记。
// 在这里修改AI的性格和回复风格
''';
}
```

## 📋 系统要求

- **Flutter SDK**: 3.0+
- **Android**: API 21+ (Android 5.0+)
- **网络**: AI功能需要联网

## ⚠️ 注意事项

1. **API密钥安全**: 内置API密钥仅用于演示，生产环境建议使用后端代理
2. **网络要求**: AI功能需要网络连接
3. **数据存储**: 笔记保存在本地，卸载应用会丢失
